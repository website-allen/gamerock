What is Mt Manager about ?

MT Manager is a very powerful file manager and editor. By default, on our andriod 9-12 we cant access certain files. However, with MT Manager the case is different we can access files that our andriod's make difficult to reach and modify, translate them easily. However, if you want to modify you device's system directory you will need root access but this isn't mostly needed, Besides there's a lot of fun to catch without root access.

Features of MT Manager

  • Effective File handling system, such as copy, move, delete. If you give root privilege to MT, you can access the system directory, remount the file system as read-write, modify file permissions and owner.
  • Open the ZIP file like WinRAR. You can rename, move, delete, add or replace external files to the ZIP without decompression and repackaging and decompress files in the ZIP.
  • Many powerful APK editing features, like arsc editor, dex editor, xml editor, hex exitor. you can also sign apk, clone apk, optimise apk, remove signature verification, anti-confuse apk resource, translate app.
  • APK Encryption
  • Text editor with syntax highlighting
  • Creating a backup (.bak)
  • Fonts viewing
  • Script Executiom
